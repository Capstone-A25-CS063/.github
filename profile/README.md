## üöÄ About the Project

**PRISM** is a predictive portal designed to improve the effectiveness of outbound calls in time deposit marketing.  
By leveraging **machine learning**, PRISM helps prioritize the most potential customers, allowing sales teams to focus on high-value prospects, reduce time spent on low-quality leads, and improve overall operational efficiency.

Developed using a **Design Thinking approach**, PRISM emphasizes usability, data-driven insights, and seamless integration into daily sales workflows to support more accurate and informed marketing decisions.

## üìä Machine Learning Overview

The machine learning component processes historical customer data to generate predictive insights on customer potential.  
Trained models are exported using **Joblib** and integrated into the application to support real-time prediction and prioritization within the marketing workflow.
